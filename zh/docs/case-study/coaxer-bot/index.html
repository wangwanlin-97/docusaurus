Case Study: Coaxer BOT

Image credit: 庆喜儿sarah

三步教你用Node做一个微信哄女友神器（不要说程序员不懂浪漫哦），不用懂代码！不用学编程！看教程，三部教你做一个微信每日说定时工具，每天定时给女朋友发暖心情话，也可以给女朋友做个智能聊天机器人！

Usage​

  • 第一步链接地址：node官网 https://nodejs.org/zh-cn/
  • 第二步链接地址：代码地址 https://github.com/gengchen528/wechatBot

微信每日说是基于 node 与wechaty的微信小情话工具。最初功能只有每日发送天气和一句情话，后来添加了智能机器人聊天功能，自动加群，自动加好友，定时助手功能等。

但由于本项目面向小白用户与刚接触 node 开发的用户，故拆分了两个项目，一个是功能专一面向小白的《微信每日说》（也就是本项目），另一个也在 Leo 的仓库下《智能微秘书》面向有较多编程经验的用户。

Features​

  1. 定时给女朋友发送每日天气提醒，以及每日一句
  2. 天行机器人自动陪女朋友聊天（需要自己申请天行机器人api，不过目前开源的机器人 api 都不要抱太大希望，因为很傻的，如果你有发现好的机器人可以来推荐）
  3. 垃圾分类功能，使用方法：？垃圾名称
  4. 最近看到 python 版支持多女朋友配置，我思考了一下，还是不要加了比较好，我们要做一个专一的人，哈哈
  5. 想要更多群管理，自动加好友功能，并体验在线配置服务，请移步《智能微秘书》

If you understand English, this part is for you!​

Image credit: 庆喜儿sarah

Three steps to teach you to use Node to make a WeChat girlfriend artifact (don’t say that programmers don’t understand romance), don’t need to understand the code! No need to learn programming! Look at the tutorial, the third part teaches you to make a WeChat daily talk timer tool, to send your girlfriend a warm message every day, and you can also be an intelligent chat robot for your girlfriend!

Usage​

-The first step link address: node official website https://nodejs.org/zh-cn/ -The second step link address: code address https://github.com/gengchen528/wechatBot

WeChat Daily Talk is a WeChat little love talk tool based on node and wechaty. Initially, the function was only to send the weather and a sentence of love every day. Later, the intelligent robot chat function was added, automatically adding groups, automatically adding friends, and timing assistant functions.

However, because this project is for Xiaobai users and users who are new to node development, two projects have been split, one is the function-specific "WeChat Daily Talk" (that is, this project) for Xiaobai, and the other is also "Smart Secretary" under Leo's warehouse is for users with more programming experience.

Features​

  1. Send a daily weather reminder to your girlfriend regularly, as well as a daily sentence
  2. Tianxing robot automatically chats with girlfriend (you need to apply for Tianxing robot api by yourself, but don’t have too much hope for the current open source robot api, because it is silly, if you find a good robot, you can recommend it)
  3. Garbage classification function, how to use it:? Garbage name
  4. Recently I saw that the python version supports multiple girlfriend configurations. I thought about it. It's better not to add it. We should be a dedicated person, haha
  5. If you want more group management, automatic friend addition function, and experience online configuration service, please move to "Smart Micro Secretary"
